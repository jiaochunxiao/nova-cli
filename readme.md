## Nova-Cli

一个用于快速创建项目的CLI工具

### 命令行指令

```
> nova --help
Usage: nova [options] [command]

Options:
  -V, --version    output the version number
  -i, --info       output some info of the tools
  -h, --help       output usage information

Commands:
  init <app-name>  create a new project from some template projects
```
